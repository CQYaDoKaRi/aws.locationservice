# Amazon Location Service 

## インストール  
```
npm install
```

## 起動  
- 起動
    ```
    npm run start
    ```

- 起動（開発モード）  
    ```
    npm run start-dev
    ```

## 開発
- formatter & lint
    ```
    npm run check
    ```

- formatter
    ```
    npm run formatter
    ```

- lint
    ```
    npm run linter
    ```

- ビルド
    ```
    npm run build
    ```
